在demo中编写Application并启动

package com.b4.mcr;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class McrAuthDemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(McrAuthDemoApplication.class, args);
    }
}

spring:
  session:
    store-type: none

spring boot会提供一些默认的spring security配置，所以在进行访问api时会进行拦截

在章节开始，笔者不会直接去讲spring security的内容，而会去说一些Web的相关知识，所以把spring boot提供的配置禁用一下

application.yml

security:
  basic:
    enabled: false
